# IOT POTENTIAL

### Installation & Run

```bash
git clone git@github.com:Vinnie1986/iotpotential.git
cd Flask-GoogleMaps
python setup.py install
python setup.py develop
cd webapp
python main.py
```

Access: http://localhost:5000/
